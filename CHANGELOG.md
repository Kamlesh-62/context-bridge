# Changelog

### Added
- 3 simple MCP tools: `remember`, `recall`, `memory`
- Markdown-based storage (`.ai/memory/` directory with individual `.md` files)
- Human-readable frontmatter format
- Keyword and tag-based search with relevance scoring
- Support for Claude Code, Codex CLI, and Gemini CLI
- Full test suite with vitest
- TypeScript with strict mode
- ESLint + Prettier code quality
- CI/CD with GitHub Actions
